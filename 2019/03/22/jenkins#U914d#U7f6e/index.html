jenkins配置

  1. jenkins地址：/usr/share/jenkins/jenkins.war
  2. 请在docker安装时 装最新的 ！！！！！
  3. docker pull jenkins/jenkins:版本号
  4. jenkins启动后需要进行配置：
    1. 配置JAVA_HOME，这需要你docker的jenkins镜像存在JAVA_HOME，在jenkins中可以设置目录
    2. 配置MAVEN_HOME，同上，使用apt命令下载后指定目录

jenkins配置部署测试 （后端）
